Belajar CSS

Apa Itu CSS?

CSS atau Cascading Style Sheet adalah dokumen web yang berfungsi untuk mengatur tampilan dokumen web lain, seperti HTML. CSS digunakan untuk mengatur aspek visual, seperti warna latar belakang, gaya huruf, jarak antar elemen, dan lain-lain. Meskipun CSS tidak menggantikan HTML, ia memainkan peran penting dalam membuat tampilan dokumen web menjadi lebih menarik dan dinamis. Ini adalah contoh dokumen CSS.

Bagaimana Menggunakan CSS?

Ada tiga cara menggunakan CSS:

  1. Inline Styles: Menyisipkan kode CSS langsung ke dalam tag yang ingin diubah gaya-nya.
  2. Internal/Embedded Styles: Menyisipkan kode CSS di dalam bagian <head> dokumen HTML.
  3. Eksternal/Linked Styles: Menyisipkan kode CSS pada file terpisah dan menghubungkannya dengan tag <link> di dokumen HTML.
